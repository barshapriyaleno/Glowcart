About GlowCart

GlowCart is a student startup-based investigatory project created for the academic year 2025–26. The aim of this project is to understand the basics of web development and digital business models.

This website is developed using HTML, CSS, and JavaScript and focuses on a clean pastel-themed user interface for cosmetic products.

In the future, GlowCart can be expanded into a complete e-commerce platform with payment gateways and user accounts.
